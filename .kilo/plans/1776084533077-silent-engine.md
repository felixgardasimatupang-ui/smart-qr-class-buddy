# Plan: Mobile Version - PWA Setup

## Goal
Membuat versi mobile yang dapat diinstal dan diakses seperti aplikasi native di smartphone.

## Current State
- App sudah memiliki `MobileLayout.tsx` dengan bottom navigation untuk student/parent
- Responsive design sudah ada (Tailwind)
- Belum ada PWA configuration

## Steps to Implement

### 1. Install PWA Plugin
```bash
npm install vite-plugin-pwa -D
```

### 2. Configure PWA in vite.config.ts
- Add manifest.json configuration
- App name: "Smart QR Class"
- Theme color: cyan-600 (#0891b2)
- Icons: 192x192, 512x512
- Standalone display mode

### 3. Update index.html
- Add PWA meta tags (theme-color, apple-mobile-web-app-capable, etc.)
- Update title to "Smart QR Class Buddy"

### 4. Add PWA Icons
- Create simple SVG icons or use placeholder
- Add to public/ folder

### 5. Test & Deploy
- Run `npm run build`
- Deploy ke Vercel (sudah ada panduan di PANDUAN_DEPLOY.md)

## Mobile-Specific Enhancements (Optional)
- Safe area padding untuk notch devices
- Touch target optimizations

## Deployment URL
Setelah deploy, akses URL di mobile → "Add to Home Screen" → bisa digunakan offline.
